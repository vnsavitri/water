# 💧 Water Tracker

A minimal, mobile‑friendly water tracking web app designed for dark mode and daily use on iPhone (and other devices). Track how many glasses you drink, get visual feedback, and celebrate when you hit your goal.

## Features

- Dark mode UI with soft gradients and large tap targets  
- Circular progress ring with goal and over‑goal states  
- Visual glass grid showing filled vs remaining glasses  
- Smart status messages (start, halfway, goal reached, overachiever)  
- Confetti celebration and optional haptic feedback on goal  
- Daily stats: goal completion % and total volume in liters  
- Automatic daily reset and session persistence

## Usage

1. Open `index.html` in a browser or deploy it (e.g. GitHub Pages / Netlify).
2. Tap **+ Add Glass** each time you drink a glass (250 ml).
3. Use **Remove** to correct mistakes.
4. Use **Reset Daily Count** to start fresh for the day.

## Tech Stack

- HTML
- CSS (no framework)
- Vanilla JavaScript

## Deployment

You can host this as a static site on any provider (e.g. GitHub Pages or Netlify).  
Just serve the `index.html` file from the repo root.
